## 3. ON-PAGE SEO OPTIMIZATION

### Technical SEO Checklist

#### Page Title Tags
**Format:** "[Form Name] Form Download & Auto-Fill Guide | [Site Name]"

**Examples:**
- "Aadhaar Address Update Form Download & Auto-Fill Guide | FormEase"
- "PAN Card Correction Form: Complete Guide & Download | FormEase"
- "Passport Size Photo Maker Online Free | FormEase"

**Requirements:**
- 50-60 characters
- Include primary keyword
- Include brand name
- Unique for each page

#### Meta Descriptions
**Format:** "Learn how to fill [form name] form. Download PDF, required documents, eligibility, fees, step-by-step guide. Auto-fill form online in 2 minutes!"

**Requirements:**
- 150-160 characters
- Include primary keyword
- Compelling call-to-action
- Unique for each page

#### H1 Tags
- One H1 per page (main heading)
- Include primary keyword
- Clear and descriptive

#### H2/H3 Tags
- Use for section headings
- Include relevant keywords
- Hierarchical structure (H1 → H2 → H3)

#### URL Structure
**Format:** `/[form-name]-form-guide` or `/guides/[form-name]`

**Examples:**
- `/guides/aadhaar-address-update`
- `/guides/pan-correction`
- `/tools/passport-photo-maker`

**Requirements:**
- Short and descriptive
- Include primary keyword
- Use hyphens (not underscores)
- Lowercase

#### Internal Linking Strategy
- Link between related guides
- Link to tools from guides
- Link to forms from guides
- Use descriptive anchor text
- Aim for 3-5 internal links per page

#### Image Optimization
- Use descriptive file names: `aadhaar-address-update-form-example.jpg`
- Add alt text: "Aadhaar address update form filled example"
- Compress images (reduce file size)
- Use WebP format when possible
- Add captions where relevant

#### Schema Markup
Add structured data for:
- **Article schema** (for guides)
- **HowTo schema** (for step-by-step guides)
- **FAQ schema** (for FAQ sections)
- **SoftwareApplication schema** (for tools)

**Example HowTo Schema:**
```json
{
  "@context": "https://schema.org",
  "@type": "HowTo",
  "name": "How to Fill Aadhaar Address Update Form",
  "step": [
    {
      "@type": "HowToStep",
      "text": "Download the form from official website"
    }
  ]
}
```

## 6. TECHNICAL SEO

### Site Speed Optimization
- Optimize images (WebP, compression)
- Enable caching
- Minify CSS/JavaScript
- Use CDN (Vercel provides this)
- Lazy load images
- Reduce server response time

**Target:**
- PageSpeed Score: 90+ (mobile & desktop)
- First Contentful Paint: < 1.5s
- Time to Interactive: < 3s

### Mobile Optimization
- Responsive design (mobile-first)
- Touch-friendly buttons
- Readable font sizes
- Fast mobile page speed
- No horizontal scrolling

### Site Structure
```
/ (Home)
├── /forms
│   ├── /aadhaar-address-update
│   ├── /pan-correction
│   └── ...
├── /guides
│   ├── /aadhaar-address-update
│   ├── /pan-correction
│   └── ...
├── /tools
│   ├── /passport-photo-maker
│   ├── /pdf-merge
│   └── ...
└── /blog (optional)
```

### XML Sitemap
- Generate sitemap.xml
- Include all pages
- Submit to Google Search Console
- Update monthly (automated)

### Robots.txt
```
User-agent: *
Allow: /
Disallow: /admin
Disallow: /api
Sitemap: https://yoursite.com/sitemap.xml
```
